# 🚗 안전운전 리포트 대시보드

> DTG 데이터를 기반으로 11대 위험운전행동을 분석하고 유류비 절감액과 CO2 감축 효과를 자동으로 계산하는 안전운전 리포트 대시보드

## 📋 프로젝트 개요

이 프로젝트는 한국교통안전공단 AI청년자문단 활동의 일환으로 개발된 안전운전 분석 시스템입니다. 차량의 DTG(Digital Tachograph) 데이터를 분석하여 운전 습관을 점검하고, 경제적·환경적 효과를 시각화하여 제공합니다.

## ✨ 주요 기능

### 📊 데이터 분석
- **CSV/XLSX 파일 업로드**: DTG 데이터 손쉽게 import (두 형식 모두 지원)
- **11대 위험운전행동 분석**: 과속, 급가속, 급감속 등 상세 분석
- **차량별 위험도 등급**: A~E 등급으로 안전 운전 평가

### 💰 경제적 효과
- **유류비 절감액 계산**: 안전운전으로 절감되는 기름값 추정
- **통계 기반 알고리즘**: 실제 주행 데이터 기반 정확한 계산

### 🌱 환경적 효과
- **CO2 감축량 계산**: 친환경 운전의 환경 기여도 측정
- **나무 심재 효과**: 감축된 CO2를 나무 몇 그루에 해당하는지 시각화

### 📈 시각화
- **인터랙티브 차트**: Recharts를 활용한 동적 그래프
- **상세 리포트**: 차량별, 항목별 종합 분석 제공

### 📱 반응형 디자인
- **모바일 최적화**: 스마트폰, 태블릿, 데스크톱 완벽 지원
- **적응형 레이아웃**: 화면 크기에 따른 최적의 UI 제공
- **터치 친화적**: 모바일 환경에서 편리한 조작 경험
- **카드/테이블 전환**: 모바일에서는 카드, 데스크톱에서는 테이블 형태

### 프론트엔드
- **React 19** ⚛️ - 최신 버전의 컴포넌트 기반 개발
- **TypeScript** 📘 - 타입 안정성 확보
- **Tailwind CSS v4** 🎨 - 모던한 디자인 시스템
- **Vite** ⚡ - 빠른 빌드 및 개발 환경

- **PapaParse** 📄 - CSV 데이터 파싱

## 🚀 시작하기

### 설치 및 실행

```bash
# 레포지토리 클론
git clone https://github.com/psw204/safety-dashboard.git
cd safety-dashboard

# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build
```

### 환경 요구사항
- Node.js 18.0 이상
- npm 9.0 이상

## 📁 파일 형식

DTG 데이터는 다음 형식의 CSV 또는 XLSX 파일로 제공되어야 합니다:

### CSV 형식
```csv
인덱스,차량번호,총운행거리(km),합계,과속,장기과속,급가속,급출발,급감속,급정지,급좌회전,급우회전,급유턴,급앞지르기,급진로변경
```

### XLSX 형식
- 첫 번째 시트에 데이터 포함
- 첫 번째 행은 헤더(컬럼명)
- CSV와 동일한 컬럼명 구조

### 기술적 성취
- **다중 파일 형식 지원**: CSV와 XLSX 파일 모두 처리 가능한 파싱 시스템
- **데이터 처리 파이프라인**: CSV/XLSX → 파싱 → 분석 → 시각화 일관된 흐름

- **타입 안정성**: TypeScript로 런타임 에러 최소화

### 비즈니스 가치
- **안전운전 문화**: 데이터 기반의 객관적 운전 습관 개선
- **비용 절감**: 유류비 절감 효과로 경제적 인센티브 제공
- **ESG 경영**: CO2 감축으로 환경적 책임 이행
